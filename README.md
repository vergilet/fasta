Fasta is a mini web framework

## Installation

```bash
 $ gem install fasta
```

```bash
 $ fasta plz NEW_APP_NAME
```

```bash
 $ cd NEW_APP_NAME
```

## Start

```bash
 $ fasta go
```

or advanced
```bash
 $ puma boot.ru -e production -p 3210
```
